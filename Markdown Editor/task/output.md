### Description
This is a markdown editor with limited functionality made entirely with **Python**
### Functions:
##### Available Formatters:
1. plain
2. bold
3. italic
4. header
5. link
6. inline-code
7. new-line
8. ordered-list
9. unordered-list
##### Special Commands
1. !help
2. !done
